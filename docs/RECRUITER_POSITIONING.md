# Recruiter Positioning Guide

## Pitch de 30 segundos (PT-BR)
Este projeto mostra como eu organizo um fluxo analítico completo: carga de dados, limpeza, validação de qualidade, dashboard em Streamlit e persistência em SQLite. O foco é transformar dados brutos em informação útil para decisão de negócio, com documentação e testes para dar previsibilidade ao processo.

## 30-second pitch (EN)
This project shows how I build a practical analytics workflow: data ingestion, cleaning, quality validation, Streamlit dashboard delivery, and SQLite persistence. The goal is to turn raw data into business-ready insights with documentation and testing for predictable execution.

## Explicacao simples do projeto
Aplicacao de portfolio para analise de dados com:
- ingestao de CSV/XLSX
- curadoria automatica
- score de qualidade da base
- visualizacoes e insights em dashboard
- persistencia opcional com metadados de governanca e privacidade

## Competencias demonstradas
- analise de dados orientada ao negocio
- limpeza e padronizacao de dados
- qualidade de dados e rastreabilidade
- documentacao tecnica de apoio
- dashboarding com Streamlit
- boas praticas de testes e CI

## Cargos compativeis
- Analista de Dados Junior
- Analista de Dados
- Analista de BI Junior
- Analista de BI
- Analytics Engineer inicial/intermediario
- projetos freelance de dados

## Como falar do projeto em entrevista
- Comece pelo problema de negocio.
- Explique o fluxo ponta a ponta em 4-5 etapas.
- Mostre como o `Quality Score` orienta a confianca da analise.
- Cite onde aplicou documentacao, testes e controles basicos de governanca.

## Como responder: "Qual foi seu papel?"
Eu desenvolvi o projeto de ponta a ponta: modelagem do fluxo, implementacao do dashboard, curadoria dos dados, testes, documentacao e publicacao da demo. Tambem defini as regras de qualidade e os controles basicos de privacidade usados no app.

## Como responder: "Esse projeto e real ou portfolio?"
E um projeto de portfolio com problema realista e dados de referencia. O objetivo e demonstrar capacidade tecnica e organizacao analitica aplicando boas praticas que tambem sao usadas em ambientes reais.

## Como responder: "Voce aceita vaga junior?"
Sim. Estou aberto a vagas Junior e plenas iniciais em Dados e BI, com foco em aprender rapido, entregar com consistencia e evoluir com o time.
