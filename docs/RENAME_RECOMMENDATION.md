# Repository Rename Recommendation

## Current name
- `data-senior-analytics`

## Suggested names
- `data-analytics-workflow` (recommended)
- `analytics-dashboard-workflow`
- `data-quality-dashboard`
- `business-analytics-workflow`
- `analytics-portfolio-dashboard`

## Best option
`data-analytics-workflow` is the best option because it is neutral, professional, easy to understand, and does not position the project around seniority.

## Expected impact
- Better alignment with Junior Data Analyst, Data Analyst, BI Analyst, and early/intermediate Analytics Engineer opportunities.
- Clearer project scope for recruiters and tech leads.
- Less risk of overpromising in portfolio positioning.

## GitHub rename checklist
1. Open repository `Settings`.
2. Go to `General`.
3. Update `Repository name` to `data-analytics-workflow`.
4. Save changes and confirm redirect is active.
5. Review badges and documentation links.
6. Update pinned repositories and profile README references.
7. Validate Streamlit Cloud configuration if connected to repo name.
